#  Social Media Ad Campaign Performance Dashboard

An interactive **Power BI dashboard** built to analyze and monitor social media advertising performance across multiple marketing channels. This project helps marketing teams track ROI, CTR, CPC, CPA, revenue trends, and campaign efficiency to make data-driven decisions.

---

##  Project Overview

This dashboard consolidates campaign data from platforms such as **Google Ads, Instagram Ads, Email Campaigns, Influencer Marketing, and Referrals** to provide a unified performance view.

It enables stakeholders to:

* Identify high-performing channels
* Track revenue growth trends
* Optimize ad spend based on ROI and CPA
* Monitor campaign effectiveness in real time

2. Open the `.pbix` file in **Power BI Desktop**.
3. Refresh data if needed.
